크기는 font-size속성으로 변경할 수 있고 기본 크기는 p태그의 글자 크기로 동일하다
h1태그의 글자는 이미 200%이 된 크기로 출력이 되고 있고
동일한 font-size를 넣어주면 다른 글자들도 동일한 크기가 된다

html 태그의 전체 크기 구하기

안녕하세요

태그의 크기를 구하는 방법 height/width : 콘텐츠의 크기
padding : 패딩의 크기
border : 테두리의 크기
margin : 마진의 크기
콘텐츠 + 패딩 + 테두리 + 마진 = 전체 크기

box-size border-box변경

box-size를 border-box로 변경을 하고 작성하면 height/width의 값이 border + padding + content 값으로 바뀐다
크기값이 중요한 경우에는 border-box로 바꿔서 작성
